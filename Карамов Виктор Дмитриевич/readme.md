# Личный блог на PHP

Полнофункциональное веб-приложение для ведения личного блога с современным интерфейсом и всеми необходимыми функциями.


Аутентификация
- ✅ Регистрация новых пользователей
- ✅ Вход в систему
- ✅ Безопасный выход
- ✅ Защита сессий с таймаутом

Управление постами
- ✅ Создание новых постов (только авторизованные пользователи)
- ✅ Редактирование собственных постов
- ✅ Удаление собственных постов
- ✅ Просмотр всех постов на главной странице
- ✅ Детальный просмотр отдельного поста

Комментарии
- ✅ Добавление комментариев к постам
- ✅ Удаление собственных комментариев
- ✅ Просмотр всех комментариев к посту

Взаимодействие
- ✅ Система лайков/дизлайков (только авторизованные пользователи)
- ✅ Подсчет просмотров постов
- ✅ AJAX-обновление реакций без перезагрузки страницы

Аналитика
- ✅ Статистика профиля пользователя
- ✅ Количество постов, просмотров, лайков, дизлайков
- ✅ История комментариев пользователя

Стек

- **Backend**: PHP 8+
- **База данных**: MySQL
- **Frontend**: HTML5, Tailwind CSS, JavaScript
- **Безопасность**: PDO, CSRF-защита, XSS-защита
- **Иконки**: Font Awesome

Требования

- PHP 8.0 или выше
- MySQL 5.7 или выше
- Веб-сервер (Apache/Nginx)

 Установка

1. 

```bash
# Скопируйте файлы в папку вашего веб-сервера
# Например, для XAMPP: C:\xampp\htdocs\blog
```

2. 
  1. Создайте базу данных MySQL:
```sql
CREATE DATABASE blog_db CHARACTER SET utf8mb4 COLLATE utf8mb4_unicode_ci;
```

  2. Импортируйте структуру базы данных:
```bash
mysql -u root -p blog_db < database.sql
```

  3. Настройка конфигурации

Отредактируйте файл `src/config.php` для настройки подключения к базе данных:

```php
define('DB_HOST', 'localhost');
define('DB_NAME', 'blog_db');
define('DB_USER', 'root');
define('DB_PASS', 'your_password');
```

  4. Запуск приложения

Откройте браузер и перейдите по адресу:
```
http://localhost/blog/public/
```

Структура базы данных

Таблица `users`
- `id` - Уникальный идентификатор
- `login` - Логин пользователя
- `password_hash` - Хеш пароля
- `created_at` - Дата создания

Таблица `posts`
- `id` - Уникальный идентификатор
- `user_id` - ID автора поста
- `title` - Заголовок поста
- `content` - Содержание поста
- `created_at` - Дата создания
- `updated_at` - Дата обновления
- `likes` - Количество лайков
- `dislikes` - Количество дизлайков
- `views` - Количество просмотров

Таблица `comments`
- `id` - Уникальный идентификатор
- `post_id` - ID поста
- `user_id` - ID автора комментария
- `text` - Текст комментария
- `created_at` - Дата создания

Таблица `post_reactions`
- `id` - Уникальный идентификатор
- `post_id` - ID поста
- `user_id` - ID пользователя
- `reaction_type` - Тип реакции (like/dislike)
- `created_at` - Дата создания


После установки вы можете:

2. Создать свой первый пост
3. Добавить комментарии к постам
4. Оценить посты лайками/дизлайками
5. Просмотреть статистику в профиле

Настройка

### Изменение настроек сайта
Отредактируйте константы в `src/config.php`:
```php
define('SITE_NAME', 'Название вашего блога');
define('SITE_URL', 'http://your-domain.com');
define('POSTS_PER_PAGE', 10); // Количество постов на странице
```
